## Getting Started

### Prerequisites

Just a modern web browser. No build tools required.

### Run Locally

```bash
git clone https://github.com/golusourav72-commits/Personal-Portfolio-Website.git
cd Personal-Portfolio-Website
```

Open `index.html` directly in your browser, or serve it locally:

```bash
# Using Python
python3 -m http.server 5500

# Using VS Code Live Server extension
# Right-click index.html → "Open with Live Server"
```

Then visit `http://localhost:5500`

## Deployment

This site is deployed using **GitHub Pages**:

1. Push changes to the `main` branch
2. Go to repo **Settings → Pages**
3. Set source branch to `main` (root)
4. Site auto-publishes at `https://golusourav72-commits.github.io/Personal-Portfolio-Website/`
